# FraudShield AI (adshield-dashboard3)

FraudShield AI is a high-performance, full-stack cybersecurity platform designed for Ad Fraud Detection and Identity Verification.

## Features
- **Ad Fraud Analytics**: Real-time detection and visualization of fraudulent ad activity.
- **Identity Verification (KYC)**: Seamless user identity verification with progress tracking.
- **Modern UI**: Built with Next.js, Tailwind CSS, and shadcn/ui featuring a premium dark-mode aesthetic.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
